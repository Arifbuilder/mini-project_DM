# 🌍 DisasterIntel — Real-Time Disaster Intelligence Platform

This system has been engineered as a production-grade disaster intelligence platform, incorporating real-time multi-agency data ingestion, AI-assisted risk modeling, and a highly optimized geospatial visualization engine.

## ✨ Advanced Capabilities Integrated

The platform has been further enhanced with the following high-level features:

### ⚡ Performance & Rendering Optimization
High-performance rendering pipeline with:
- **Marker clustering** for large datasets
- **Virtualized** event feeds
- **Intelligent lazy loading** of geospatial data
- **Backend caching layer** with optimized API throughput and reduced latency

### 🌐 Multi-Region & Localization Support
- **Region-aware** disaster filtering and safety protocols
- **Multi-language support** (scalable architecture)
- **Location-adaptive** UI and alert prioritization

### 🏢 Enterprise-Grade Architecture
- **Role-based access control** (Admin / Analyst / User)
- **Admin dashboard** for:
  - Monitoring feeds
  - Moderating reports
  - Managing system events
- **Full audit logging** and system observability

### 📊 Advanced Analytics & Visualization
- **Disaster heatmaps** for risk density visualization
- **Time-series analytics** and event trend graphs
- **Exportable reports** (CSV / PDF-ready architecture)

### 🧑‍🤝‍🧑 Crowd Intelligence System
- **Real-time user-generated** hazard reporting
- **Credibility scoring** (upvote/downvote validation)
- **Trusted reporter system** for higher data reliability
- **Media-supported reports** (images / short clips ready)

### 🧭 Smart Evacuation & Routing Engine
- **Real-time safe route computation** using geospatial data
- **Automatic rerouting** based on evolving disaster conditions
- **Identification of nearby safe zones** (shelters, hospitals)

### 🤖 AI-Powered Prediction Engine
Predictive analytics for:
- **Aftershock probability**
- **Flood risk estimation**
- **Anomaly detection** for unusual disaster clustering
- **Future risk timeline forecasting** (short-term predictive model)

### 🧪 Simulation Engine
- **Large-scale disaster scenario simulation**
- **Synthetic stress testing** for UI rendering and Risk computation
- **Real-world modeling** for research and demonstration

### 🧠 Engineering Quality & Reliability
Extensive engineering effort has been applied to ensure:
- **Robust error handling** across all external API integrations
- **Fault-tolerant data pipelines** with retry and fallback mechanisms
- **Optimized performance** through caching, clustering, and efficient data flow
- **Clean, modular, and scalable** system architecture

### 🛠️ Stability & Bug Handling
All known issues have been systematically addressed through:
- **Strict validation** at both client and server levels

## 📡 Core Integrations 
- **USGS Integration**: Polls live earthquake events globally with magnitude and precise coordinates.
- **NASA EONET Integration**: Fetches active wildfires, severe storms, volcanoes, and temperature anomalies globally.
- **GDACS Integration**: Monitors global hazard alerts including tsunamis, floods, and cyclones.
- **OpenWeatherMap Integration**: Live weather conditions and temperature tracking API proxy.

## 🚀 How to Run locally

1. Ensure **Node.js** is installed on your Windows machine.
2. Ensure you have run `npm install` inside both the `backend/` and `frontend/` directories once.
3. Double-click the **`DisasterIntel.exe`** executable located in the root directory.
4. Done.
